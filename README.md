# DueProcessAI123

## Deployment Instructions

This is a Next.js app. To deploy:

1. Install dependencies:
   ```bash
   npm install
   ```
2. Build the app:
   ```bash
   npm run build
   ```
3. Start the app:
   ```bash
   npm start
   ```

For Vercel deployment, just connect your repo and Vercel will auto-detect Next.js.
